# Error Memory — README

This folder is long-term memory for errors and their solutions. See [[../../main.md]] section 2 for the full protocol.

**Before fixing any error:** search this folder for a matching symptom/tag.
**After fixing any error:** log it here immediately, in the file matching its topic area (create a new topic file if none fits).

## Files in this folder
- One `.md` file per topic/area (e.g. `build.md`, `api.md`, `deploy.md`)
- Each entry uses this format:

```md
### [ERROR] short descriptive title
**Context:** where/when this happens
**Symptom:** exact error message or behavior
**Root cause:** why it happens
**Solution:** what fixed it
**Tags:** keywords for matching
```

Don't log one-off typos. Do log anything that cost real debugging time or is likely to recur.
